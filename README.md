Shared prettier configuration.

## Usage

1. Install the package

```sh
npm install --save-dev @arunnalla/prettier-config
```

2. Edit `package.json`

```json
{
  // ...
  "prettier": "@arunnalla/prettier-config"
}
```

References:

- https://prettier.io/docs/en/configuration.html#sharing-configurations
